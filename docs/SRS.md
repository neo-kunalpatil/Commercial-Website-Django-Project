# Requirements

## Functional Requirements
- User registration, login, logout
- Seller registration and seller profile
- Role-based access (buyer/seller/admin)
- Product posting by seller
- Product browsing by buyer
- Category and search system
- Product detail page with description, price, rating, image
- Cart and buy now functionality
- Checkout and payment process
- Receipt/invoice generation
- Order history
- Order tracking
- Return request
- Reviews and ratings

## Non-Functional Requirements
- Responsive user interface
- Secure user authentication
- Reusable templates
- Clean project structure
- Proper form validation
- Maintainable code
- Scalable module design

## Software Requirements
- Python
- Django
- SQLite for development
- Bootstrap
- HTML, CSS, JavaScript
- VS Code
- Git and GitHub

## Python Packages
- Django
- Pillow
- djangorestframework
- django-filter
- reportlab
- python-dotenv
- razorpay (if payment gateway used)
- celery and redis (optional, later stage)

## Hardware Requirements
- Laptop/Desktop
- Internet connection
- Minimum 4 GB RAM
- Browser
